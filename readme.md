# Spring Boot micro-services example
Example based on the following technologies:

* Java
* Spring Framework
* Spring Boot
* Spring Cloud
* Netflix Eureka

Versions of the technologies have used:

|Technology|Version|
|----------|-------|
|Java SE|1.8|
|Spring Boot|1.5.21.RELEASE|
|Spring|4.3.24.RELEASE|
|Spring Cloud|Edgware.SR6|
